## 搭配理论

### 1. Food Pairing Hypothesis（Ahn et al. 2011）

- **西餐倾向**：配对的食材 **共享** 大量挥发性香气化合物 → 气味"融合"（番茄+罗勒+mozzarella、草莓+黑巧克力）
- **亚洲菜倾向**：配对的食材 **很少** 共享香气化合物 → 气味"对比"（酱油+姜+芝麻油 各走各的）
- 结论：没有"universal"配对规则——跨菜系用同一套会违和

### 2. 五味平衡

- **酸**（柠檬、醋、葡萄酒、酸奶）→ 提亮、切腻
- **甜**（糖、蜂蜜、水果）→ 平衡酸辣、让苦味后退
- **苦**（咖啡、黑巧、深焦）→ 厚度和复杂感
- **咸**（盐、酱油、鱼露、豉油）→ 放大所有风味
- **鲜/Umami**（蘑菇、番茄、奶酪、味增、酱油、海产）→ 深度

一道菜**至少命中三种**——太单一会"空"。

### 3. Umami × Umami（协同）

多种 umami 食材叠加，鲜度不是相加而是相乘：
- 谷氨酸（番茄、奶酪、酱油）+ 肌苷酸（猪牛鸡鱼）→ 3-8 倍
- 谷氨酸 + 鸟苷酸（香菇、蘑菇）→ 类似效果
- 例：番茄 + 奶酪 + 培根（BLT / 意面）、酱油 + 肉 + 蘑菇（日式炖菜）

### 4. 脂肪 + 酸（西餐惯用对比）

油脂铺底 → 酸来解腻：
- 橄榄油 + 柠檬 / 醋 → 沙拉基础
- 黄油 + 柠檬 → Beurre blanc、煎鱼
- 奶酪 + 葡萄酒
- 烤肉 + 咸菜 / 泡菜

### 5. 热 + 凉 / 脆 + 软（口感对比）

- 汤面 + 炸物（Tonkotsu + 叉烧脆边）
- 热蛋糕 + 冷冰淇淋
- 软 pasta + 烤 parmesan
- 红烧肉 + 脆生生的小葱丝

## 替代表

表里的替代品换进去**不会崩**，但口味略变。括号里是口感差异。

| 原料 | 替代 1 | 替代 2 | 备注 |
|---|---|---|---|
| 黄油 | 橄榄油 | 猪油 | 煎炒 OK；烘焙要按比例减 20% |
| 猪油 | 黄油 | 植物油 | 酥皮类优先猪油（起酥效果最好） |
| 五花肉 | 梅花肉 | 里脊（瘦一些） | 红烧类首选五花；炒菜用梅花 |
| 牛腩 | 牛腱 | 牛肩 | 都耐炖 |
| 鸡胸 | 鸡腿肉（去骨） | 火鸡胸 | 炒用鸡腿比鸡胸更嫩 |
| 基围虾 | 白虾 | 对虾 | 大小对应调整时间 |
| 老抽 | 生抽 + 少许糖色 | 日式たまり | 只图上色用糖色即可 |
| 生抽 | Tamari（无麸质） | 鱼露（咸鲜更冲） | 鱼露咸度高 20% |
| 料酒 | 米酒 | 黄酒 / 雪莉酒 | 去腥 |
| 米醋 | 苹果醋 | 白葡萄酒醋 | 酸度接近 |
| 陈醋 | 香醋 + 一滴老抽 | Balsamic（甜一些） | 味厚 |
| 豆瓣酱 | 甜面酱 + 辣椒粉 | 韩式 gochujang（更甜） | 无完美替代 |
| 花椒 | 胡椒 + 芹菜籽 | Sichuan pepper oil | 麻味无法替代，只能近似 |
| 葱 | 韭黄（嫩） | 青蒜苗 / 大葱 | 生吃只能葱 |
| 姜 | 姜粉 1/4 量 | galangal（东南亚） | galangal 更辛 |
| 蒜 | 蒜粉 1/4 量 | 细香葱（口感不同） | 蒜无完美替代 |
| 鸡汤 | 蔬菜汤 | 味增汤 + 水 | 素版 |
| 鲜奶油 | 椰浆（甜味多） | 浓缩牛奶 + 黄油 | 打发的用不了替代品 |
| 芝士丝（意餐） | 帕玛森 + mozzarella | Gruyère | 看 flavor 走向 |
| 糖 | 蜂蜜 3/4 量（减液体） | 枫糖 | 蜂蜜焦化温度更低 |
| 高筋面粉 | 中筋 + vital gluten | —— | 中筋+gluten 近似 |
| 意面（pasta） | 面条（宽） | 米粉 | 质地差异大 |
| mirin（味淋） | 米酒 + 糖 | 雪莉酒 + 糖 | 日餐甜度来源 |

## 查询工作流

用户问："X 和 Y 搭不搭？"

1. `python scripts/pairing.py --pair X Y` 看本地共现
2. 共现 ≥3 → "经典/中频搭配，本地已见"
3. 共现 <3 → 查本文
4. 本文有 → 给出依据（类别、理论、替代路径）
5. 本文也没 → 诚实说"没见过，我不确定"，建议 `fetch.py` 搜一下再定

## 查询工作流（替代）

用户问："X 能用 Y 替代吗？"

1. grep 本文的替代表 → 有则按说明
2. 替代表没有 → 按"味道 + 口感 + 用途"三维度推断：
   - 味道接近（咸/甜/酸/鲜/苦）
   - 口感接近（脆/软/嫩/韧）
   - 用途（调味/主料/点缀）
3. 还是不放心 → 建议用户先少量尝试，再定配比
