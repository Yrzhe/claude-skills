# 烹饪技法参考

一本迷你字典。walkthrough 里出现术语时，agent 从这里回答"那是什么意思"而不是去谷歌。

## 中餐核心技法

### 火候

| 名字 | 温度 | 表现 | 适用 |
|---|---|---|---|
| 温油 / 五成热 | ~140°C | 轻微烟雾，入筷子周围小泡 | 炒糖色、滑油、嫩肉下锅 |
| 热油 / 六七成热 | ~180°C | 明显烟，筷入马上起密泡 | 爆炒、滑炒 |
| 猛火 / 旺火 | 锅红 | 锅壁发白光、油面冒大烟 | 爆炒、爆锅、镬气 |
| 小火 | 微微冒泡 | 冒细小气泡，水面稳定 | 炖、焖、慢煨 |
| 微火 / 文火 | 几乎不冒泡 | 保温状态 | 熬汤、煨 |

经验规则：**肉断生改小、炒到香改旺、收汁改猛、起锅前勾芡改小。**

### 常见技法动词

- **爆**：极高温快速翻炒，不超过 30 秒（宫保鸡丁、爆炒腰花）
- **炒**：中高温翻动，1-3 分钟
- **焖**：加盖小火，汤汁逐渐收（红烧肉、土豆焖鸡）
- **炖**：冷水下料，大火煮开转小火长时间（老火汤、牛腩）
- **煨**：小火极慢（粥、羊肉汤）
- **烧**：先炒后加汤汁收浓（红烧鱼、家常豆腐）
- **熘**：勾芡后淋油（糖醋熘鱼）
- **煸**：少油慢炒至食材发干（干煸豆角）
- **氽**：开水快速一过（肉片、海鲜焯水）
- **汆**：同氽，焯水断生
- **烩**：多食材混合焖煮（什锦烩菜）
- **焗**：密闭容器烤或闷煮（盐焗鸡）

### 八大菜系风味定位

| 菜系 | 基调 | 代表菜 | 关键调料 |
|---|---|---|---|
| 川 | 麻辣鲜香 | 麻婆豆腐、宫保鸡丁、水煮鱼 | 郫县豆瓣、花椒、干辣椒 |
| 粤 | 清鲜爽滑 | 白切鸡、清蒸鱼、叉烧 | 蚝油、生抽、米酒 |
| 鲁 | 咸鲜醇厚 | 葱烧海参、糖醋鲤鱼、九转大肠 | 葱姜蒜、酱油 |
| 苏 | 甜咸平衡 | 松鼠鳜鱼、狮子头、盐水鸭 | 糖、米醋、清汤 |
| 浙 | 清爽精细 | 西湖醋鱼、东坡肉、龙井虾仁 | 绍酒、酱油 |

### 刀工

- **丝**：1mm 宽细长（青椒丝、土豆丝）
- **丁**：~8mm 方块（宫保鸡丁的"鸡丁"）
- **末**：<3mm 颗粒（蒜末、葱末）
- **片**：薄 2-3mm（肉片、玉兰片）
- **块**：>2cm 大块（红烧肉、炖肉）
- **花**：表面改刀出纹路（麦穗花刀、蓑衣黄瓜）

## 西餐核心技法

### Mise en place（一切就绪）

下锅前把所有食材切好、调料量好、工具摆好。这是西餐节奏的基础。忽略它你会来回翻抽屉，锅里的东西就焦了。

### 热度术语

| 英文 | 中文 | 温度 | 说明 |
|---|---|---|---|
| Low | 小火 | ~95°C | 炖 / simmer |
| Medium | 中火 | ~150°C | Saute 日常 |
| Medium-high | 中大火 | ~190°C | 煎肉、锁汁 |
| High | 大火 | ~230°C+ | 爆炒、快烤 |
| Smoking | 冒烟 | >230°C | 炙烤牛排 |

烤箱：
- 150°C / 300°F = 慢烤（pulled pork）
- 180°C / 350°F = 日常烘焙
- 200°C / 400°F = 烤蔬菜、烤鸡
- 230°C / 450°F = 比萨、快烤
- 高档烤箱有对流（convection），实际温度约比显示低 15°C

### 技法动词

- **Sauté**：热锅少油中高温快炒（类似"爆炒"但温度略低）
- **Sear**：猛火把表面煎成褐色外壳（锁汁）
- **Braise**：先 sear 后加液小火焖（像中餐的"烧"）
- **Roast**：烤箱干热烤整块（整鸡、整肉、整根蔬菜）
- **Bake**：烤箱、不涉及大块肉（面包、蛋糕、casserole）
- **Broil**：上火炙烤（~260°C 顶部加热）
- **Poach**：80°C 左右的液体中慢煮（鸡蛋、鱼）
- **Simmer**：小火冒小泡但不滚（汤、酱）
- **Boil**：大火滚开
- **Blanch**：开水过一下再冰水激（断生、去涩）
- **Reduce**：不加盖大火收汁
- **Deglaze**：锅底有煎肉余香时加液（酒/汤）溶解褐渣做酱

### 法式五大母酱（Sauces Mères）

基础在 classic French，衍生千变万化。

| 母酱 | 基础 | 派生举例 |
|---|---|---|
| Béchamel | 牛奶 + roux | Mornay（加 cheese）→ 千层面酱 |
| Velouté | 高汤（鸡/鱼/小牛）+ roux | Supreme（加奶油）、Allemande（加蛋黄） |
| Espagnole | 棕色小牛汤 + brown roux + 番茄 | Demi-glace、Bordelaise（加红酒） |
| Tomate | 番茄 + 蔬菜 + 香料 | 意式番茄酱、Creole |
| Hollandaise | 蛋黄 + 澄清黄油 + 柠檬 | Béarnaise（加龙蒿） |

**Roux**：等重的黄油 + 面粉炒到白/金/棕三个阶段，分别对应白酱、浅棕酱、深棕酱。

### 调味原则

- **Salt early, salt often**：肉下锅前提前 40 分钟盐（干腌）或下锅即盐；不要最后才加
- **Acid for brightness**：一口菜感觉"闷"了——缺的一般是酸（柠檬汁、醋、葡萄酒）
- **Fat + salt + acid + heat** 是 Samin Nosrat 的《盐油酸热》四元素
- **Layering**：分阶段加盐（底油、主料、收汁），比最后大把更均匀

## 常见失败诊断

| 症状 | 可能原因 |
|---|---|
| 肉老柴 | 火候过大 / 过久 / 切错纹路（要垂直纹路切） |
| 炒菜出水 | 锅不够热 / 食材没沥干 / 一次下太多 |
| 糖色发苦 | 炒糖色时冒黑烟了，重来 |
| 汤发混 | 浮沫没撇 / 大火没转小火 |
| 煎肉不上色 | 肉有水分 / 油不够热 / 一直翻 → 擦干、热锅、先别动 |
| 烤箱烤干 | 时间太长 / 没覆盖 / 烤温太低拉长时间 |
| 意面发糊 | 水太少 / 没加盐 / 煮太久 |
| 米饭夹生 | 水少了 or 火太大蒸汽跑了 → 补水再焖 5 分钟 |
